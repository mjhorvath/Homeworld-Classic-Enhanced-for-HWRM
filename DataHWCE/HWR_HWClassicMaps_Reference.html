Homeworld Classic Maps Races Table

Below is a listing of the races each player must choose (based on their position) when playing each map using the Start With setting set to HW1 Map Default and the Positions setting set to Fixed. You can safely ignore this list when playing using the other modes.

If the table says "R1" choose Kushan. If "R2" choose Taidan. If "Race_Sect" choose Hiigaran.

Note that player order is determined by the "Slot" option in the game setup screen, not by the order in which people appear in the lobby.
