Background Storyline and/or Comments:

This is a small map where player-starting locations contain the only resources forcing players to actively go after each other to maintain income. Provides frantic action where initiative and Intrepidity are paramount.

*************

This map was been designed for the 'Homeworld Map Makers Primer', used as an example throughout the map making tutorial and included as a final version for reference.

This map has been designed for a FFA bash for up to 6 players.

In order to play this may, unzip it into your SIERRA\Homeworld\Multiplayer folder. It should have five folders named Revelations2 - Revelations6 plus this text file. If you have properly placed the folders, Homeworld will automatically recognize this map on startup when you select Multiplayer maps.
