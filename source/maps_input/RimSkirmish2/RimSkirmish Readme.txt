General Info:

This is a battle map, no harvesting, no researching.  You have a starting 
fleet and enough RUs to customize it to yourself.  Players are seperated 
by an asteroid field.  This is more of a concept map, to see if battles 
interest anyone.  Enjoy!

*************


In order to play this may, unzip it into your SIERRA\Homeworld\Multiplayer 
folder. It should have 1 folder named RimSkirmish2 plus this text file. 
If you have properly placed the folders, Homeworld will automatically 
recognize this map on startup when you select Multiplayer maps.
