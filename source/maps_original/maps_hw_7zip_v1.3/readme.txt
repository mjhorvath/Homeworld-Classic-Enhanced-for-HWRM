Background Storyline and/or Comments:

This is basically a battle map where the Kadeshi and Turanic raiders duke it out. There are no harvestable resources. Hope you enjoy. 


*************

This map has been designed for 1 vs. 1 play 

In order to play this may, unzip it into your SIERRA\Homeworld\Multiplayer folder. It should
have 8 folders [asteroid.dist, description.txt, dust.dist, motherships 0 and 1, tiny.dist, resourceshere, and pirates2.level] plus this text file. If you
have properly placed the folders, Homeworld will automatically recognize this map on startup
when you select Multiplayer maps.
