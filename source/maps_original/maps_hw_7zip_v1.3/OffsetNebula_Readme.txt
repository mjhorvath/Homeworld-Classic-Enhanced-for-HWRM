Created With:  Mission Man, Notepad
====================

General Info:

Medium sized map with extensive Nebulae and a central asteroid cluster, 
with dust clouds 'tween the players starting points (acsept on 2 player).
Map is at an angle to the horizontal plane.

*************


In order to play this may, unzip it into your SIERRA\Homeworld\Multiplayer 
folder. It should have folders named OffsetNebula2-OffsetNebula4 
plus this text file. If you have properly placed the folders, Homeworld 
will automatically recognize this map on startup when you select 
Multiplayer maps.
